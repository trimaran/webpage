The Trimaran LivePC is a pre-installed version of Trimaran in a Linux host. It offers convenience for those using Trimaran in an educational setting because the LivePC can be administered centrally and then distributed to students. The LivePC is also a convenient alternative to those who run into installation problems and cannot install the pre-requisite software.

The Trimaran+SUIF LivePC combines the Trimaran compiler with a SUIF frontend, thus enabling the use of SUIF high level program analysis passes with the Trimaran compiler and its rich collection of machine specific optimizations.
